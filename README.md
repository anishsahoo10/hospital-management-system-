<h1 align="center">Hospital Management System (C Project)</h1>

<p align="center">
A console-based Hospital Management System built in C for managing patients, doctor schedules, and billing using file handling and data structures.
</p>

<hr>

<h2>Overview</h2>
<p>
This project demonstrates a simple hospital management application using C programming. It allows administrators to:
</p>
<ul>
  <li>Add and view patient details</li>
  <li>Add and view doctor schedules</li>
  <li>Create and view billing records</li>
</ul>

<hr>

<h2>Features</h2>
<ul>
  <li>Console-based menu-driven interface</li>
  <li>Persistent data storage using binary files</li>
  <li>Auto-generated unique IDs for all records</li>
  <li>Clean modular structure for each operation</li>
</ul>

<hr>

<h2>Technology Stack</h2>
<ul>
  <li><b>Language:</b> C</li>
  <li><b>Concepts Used:</b> File Handling, Structures, Arrays, Functions</li>
</ul>

<hr>

<h2>File Structure</h2>
<pre>
Hospital-Management-System/
│
├── hospital.c        # Main program file
├── patients.dat      # Stores patient records
├── schedules.dat     # Stores doctor schedules
├── bills.dat         # Stores billing records
└── README.md         # Documentation
</pre>

<hr>

<h2>How to Run</h2>
<ol>
  <li>Compile the source code:</li>
  <pre><code>gcc hospital.c -o hospital</code></pre>
  <li>Run the executable:</li>
  <pre><code>./hospital</code></pre>
</ol>

<p>
Once the program starts, follow the menu options to manage patients, schedules, and billing.
</p>
